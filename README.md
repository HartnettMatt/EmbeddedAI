# EmbeddedAI

Matt Hartnett's coursework for ECEN 5003: Embedded AI.

## Repo Layout
- `labs/` – lab assignments, one folder per lab.
- `final-project/` – final project drafts, notes, and code.

## Getting Started
1. Clone the repo: `git clone <repo-url>`.
2. Enter the folder: `cd EmbeddedAI`.
3. Explore labs or the final project: `ls labs` or `ls final-project`.

## Notes
- Each lab folder includes its own README or instructions when needed.
